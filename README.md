# FrameProcThread
本项目基于Windows平台，使用OpenCV以及Vzense深度摄像头SDK，实现地铁闸机行人检测和通行判断。

本项目用于实验室开发机测试，作为一个独立的线程模块，嵌入到原有的闸机项目中，替代了原有的基于传感器的通行检测和判断。
